# Monitoring-Dashboard

Modulare Bash-Skripte zur Überwachung der Systemressourcen.

## Module
* `cpu.sh`: Zeigt die aktuelle CPU-Auslastung.
* `ram.sh`: Überwacht den Arbeitsspeicher.
* `disk.sh`: Prüft die Festplattenbelegung.
* `network.sh`: Überwacht die Netzwerkschnittstellen.
* `monitor.sh`: Das Hauptskript, das alle Module zentral aufruft.
